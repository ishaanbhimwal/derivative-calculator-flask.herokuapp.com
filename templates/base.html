About Derivative Calculator

Derivative Calculator is a web app written using Python. It uses libraries like SymPy and Matplotlib for computing the derivative of the mathematical function and plotting the graphs. Source code of the project is available in my GitHub repo. If you have any questions or ideas for improvements to the Derivative Calculator, don't hesitate to write me an e-mail.
